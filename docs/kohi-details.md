# 公費負担医療制度 詳細仕様書

## 公費負担医療の優先順位

### 優先順位の原則
公費負担医療には**明確な優先順位**があり、複数の公費が併用される場合、優先順位の高い公費から適用されます。

### 本システムで扱う公費の優先順位（高い→低い）

| 優先順位 | 法別番号 | 名称 | 分類 |
|---------|---------|------|------|
| **1位** | **21** | **精神通院医療（自立支援）** | **公費優先** |
| **2位** | **15** | **更生医療（自立支援）** | **公費優先** |
| **3位** | **16** | **育成医療（自立支援）** | **公費優先** |
| **4位** | **54** | **難病法（指定難病）** | **保険優先** |
| **5位（最下位）** | **12** | **生活保護（医療扶助）** | **保険優先** |

**重要な原則**:
- 感染症法（法別番号29）までが「公費優先」
- それ以降は「保険優先」
- 生活保護（12）は**常に最下位**

## 各公費の詳細仕様

### 1. 公費12: 生活保護（医療扶助）

#### 基本情報
- **法別番号**: 12
- **正式名称**: 生活保護法による医療扶助
- **自己負担**: **0円**（全額公費負担）
- **優先順位**: **最下位**（他の全ての公費の後）

#### 適用範囲
- **対象**: 生活保護受給者の全ての医療
- **制限**: なし（あらゆる疾患・治療が対象）

#### 併用時の動作
他の公費と併用される場合、**他の公費で適用されなかった残りの自己負担分**を生活保護が負担します。

**例**: 精神通院医療（21）+ 生活保護（12）
- 精神科の薬: 21が90%負担 → 残り10%を12が負担 → **患者負担0円**
- 風邪薬（精神疾患以外）: 21対象外 → 12が100%負担 → **患者負担0円**

#### 請求上の注意点
- レセプトに「12」が記載されていても、**実際に12が使われているとは限らない**
- 他の公費で全額カバーされる場合、12は使用されない
- **本システムへの影響**: 12の存在=生活保護受給者であることは確実

---

### 2. 公費21: 精神通院医療（自立支援医療）

#### 基本情報
- **法別番号**: 21
- **正式名称**: 自立支援医療（精神通院医療）
- **自己負担**: **10%**（月額上限あり）
- **優先順位**: **最優先**（本システム対象公費の中で）

#### 適用範囲（非常に限定的）
- **対象疾患**: 精神障害（統合失調症、うつ病、てんかん、依存症など）
- **対象医療**: 精神障害及び**当該精神障害に起因して生じた病態**に対する通院医療のみ

**重要な制限**:
- ✅ 対象: 精神疾患の治療薬（抗精神病薬、抗うつ薬など）
- ❌ 対象外: **風邪薬、胃腸薬など精神疾患と直接関係のない薬剤**
- ❌ 対象外: 入院医療

#### 併用の実例
精神科から処方された場合でも:
```
処方例: 抗うつ薬 + 風邪薬
├─ 抗うつ薬 → 21で10%自己負担
└─ 風邪薬   → 21対象外 → 12（生活保護）で100%負担
```

#### 実務上の問題点
**CSVデータの限界**:
- レセプトには「21」と「12」の両方が記載される
- しかし、**どの薬剤がどちらの公費で処理されたかはCSVには記載されていない**
- 実際の請求額は処方内容によって変動する

---

### 3. 公費15: 更生医療（自立支援医療）

#### 基本情報
- **法別番号**: 15
- **正式名称**: 自立支援医療（更生医療）
- **自己負担**: **10%**（月額上限あり）
- **優先順位**: 2位

#### 適用範囲（極めて限定的）
- **対象者**: **18歳以上**の身体障害者手帳所持者
- **対象医療**: **障害を除去・軽減する手術等の治療**に限定

**対象となる障害と治療例**:
| 障害種別 | 対象治療 |
|---------|---------|
| 視覚障害 | 白内障手術、角膜移植術 |
| 聴覚障害 | 鼓膜穿孔閉鎖術、形成術 |
| 肢体不自由 | 人工関節置換術、関節形成術 |
| 心臓機能障害 | 心臓弁置換術、ペースメーカー埋込 |
| 腎臓機能障害 | 人工透析、腎移植 |

**重要な制限**:
- ❌ 対象外: **内科的治療のみの医療**
- ❌ 対象外: 障害と無関係の疾患（風邪、胃腸炎など）
- ✅ 条件: 「治療により確実に効果が期待できる」ことが必要

#### 併用例
```
例: 腎不全患者の人工透析 + 風邪の治療
├─ 人工透析 → 15で10%自己負担
└─ 風邪の治療 → 15対象外 → 12（生活保護）で100%負担
```

---

### 4. 公費16: 育成医療（自立支援医療）

#### 基本情報
- **法別番号**: 16
- **正式名称**: 自立支援医療（育成医療）
- **自己負担**: **10%**（月額上限あり）
- **優先順位**: 3位

#### 適用範囲
- **対象者**: **18歳未満**の児童（身体障害者手帳は不要）
- **対象医療**: 身体障害を除去・軽減する手術等の治療

**対象障害**:
- 肢体不自由、視覚障害、聴覚障害、音声・言語・咀嚼機能障害
- 心臓機能障害、腎臓機能障害、小腸機能障害、その他の内臓障害

**重要な違い（更生医療15との比較）**:
| 項目 | 育成医療（16） | 更生医療（15） |
|------|---------------|---------------|
| 年齢 | 18歳未満 | 18歳以上 |
| 手帳 | 不要 | 必要 |
| 範囲 | 障害除去・軽減 | 障害除去・軽減 |

---

### 5. 公費54: 難病法（指定難病）

#### 基本情報
- **法別番号**: 54
- **正式名称**: 難病の患者に対する医療等に関する法律
- **自己負担**: **2割**（30%→20%に軽減、月額上限あり）
- **優先順位**: 4位
- **対象疾患数**: 348疾患（2025年4月現在）

#### 適用範囲
- **対象**: 指定難病及び**当該難病に付随して発生した傷病**
- **対象医療**: 診断、処方、検査、手術、入院、在宅医療管理

**重要な特徴**:
- ✅ 対象: 難病そのものの治療
- ✅ 対象: 難病に起因する合併症
- ❌ 対象外: **難病と無関係の疾患**（風邪、外傷など）

#### 併用例
```
例: パーキンソン病患者の治療 + 風邪の治療
├─ パーキンソン病の薬 → 54で2割自己負担
└─ 風邪の治療       → 54対象外 → 12（生活保護）で100%負担
```

#### 自己負担上限額管理票
- 複数医療機関での自己負担を合算管理
- 上限額に達すると、それ以降は全額公費負担

## 公費併用時の請求フロー

### 優先順位適用の実例

#### ケース1: 21（精神通院） + 12（生活保護）
```
処方: 抗うつ薬1,000円 + 風邪薬500円

【請求先】
1. 抗うつ薬（精神疾患の治療薬）
   - 21に全額（1,000円）請求
   - 患者負担: 21のルールに従い10%（上限管理あり）
   - 12との併用で最終的に患者負担0円

2. 風邪薬（精神疾患以外）
   - 21は適用外
   - 12に全額（500円）請求 → 患者負担0円

【結果】
- 患者最終負担: 0円
- 21への請求: 1,000円（抗うつ薬のみ）
- 12への請求: 500円（風邪薬のみ）+ 21の患者負担分

【重要】旭川市への請求書には「12への請求分のみ」記載
```

#### ケース2: 15（更生医療） + 54（難病） + 12（生活保護）
```
患者: 人工透析中 + パーキンソン病 + 生活保護

医療費: 透析50,000円 + パーキンソン薬10,000円

【請求先】
1. 透析費用（更生医療対象）
   - 15に全額（50,000円）請求
   - 患者負担: 15のルールに従い10%（上限管理あり）
   - 12との併用で最終的に患者負担0円

2. パーキンソン病薬（難病対象）
   - 54に全額（10,000円）請求
   - 患者負担: 54のルールに従い20%（上限管理あり）
   - 12との併用で最終的に患者負担0円

【結果】
- 患者最終負担: 0円
- 15への請求: 50,000円（透析のみ）
- 54への請求: 10,000円（パーキンソン薬のみ）
- 12への請求: 15と54の患者負担分のみ（実額は処方内容による）

【重要】旭川市への請求書には「12への請求分のみ」記載
        → 15や54に請求された分は除外される
```

## 本システムへの影響と注意点

### CSVデータの前提条件

**重要**: 本システムに投入されるCSVデータは以下の条件を満たしています:
- ✅ すでに公費12（生活保護）を含むデータのみ
- ✅ 生活保護受給者のみが記載されている
- → 生活保護判定は不要

### 本システムで必要な判定

#### 唯一の判定: 旭川市かどうか
```javascript
// 旭川市判定のみ
patient.isAsahikawa =
  ['12016010', '12012019'].includes(patient.insurerNumber) ||
  patient.address.includes('旭川市');
```

**理由**:
- 旭川市への請求書を作成するため
- 旭川市以外の生活保護受給者は除外する必要がある

### 他の公費（21, 15, 16, 54）の問題

#### 実務上の課題
CSVに21/15/16/54が記載されている患者の場合:
- **実際に旭川市（12）への請求が発生するかはレセプト内容次第**
- 例: 精神科で風邪薬のみ処方 → 21に全額請求 → 12への請求なし

#### 解決策: チェックボックス機能

**仕様**:
1. **デフォルト**: 全患者にチェックON（請求する）
2. **手動除外**: 他公費で全額カバーされる患者はチェックOFF
3. **Excel出力**: チェックONの患者のみ出力

**実装イメージ**:
```javascript
// PatientDataクラスに追加
this.isIncluded = true; // デフォルトでチェックON

// フィルタリング（Excel出力時）
const includedPatients = patients.filter(p => p.isIncluded);
```

### フラグ表示の意味

```javascript
// フラグ判定（参考情報）
patient.hasJiritsuShien = publicCodes.some(code => ['21', '15', '16'].includes(code));
patient.hasJusho = publicCodes.includes('54');
```

**フラグの用途**:
- ⚠️ 警告表示: 「この患者は他公費あり → 実レセプト確認推奨」
- 📋 参考情報: ユーザーがチェックOFF判断の材料にする

**注意**:
- フラグがあっても請求は発生する可能性あり
- フラグがなくても12への請求は確実

## まとめ: 本システムの仕様

### 重要な理解

1. **CSVデータの前提**
   - すでに公費12（生活保護）を含むデータのみ
   - 生活保護判定は不要

2. **必要な判定**
   - 旭川市判定のみ（保険者番号 OR 住所）

3. **他公費の扱い**
   - 21/15/16/54はフラグ表示のみ
   - 実際の請求有無はユーザーが判断
   - チェックボックスで手動除外

4. **デフォルト動作**
   - 全患者を請求対象（チェックON）
   - 他公費で全額カバーされる患者のみチェックOFF

### 公費請求の原則（参考）

1. **優先順位**
   - 21（精神通院） > 15（更生医療） > 16（育成医療） > 54（難病） > 12（生活保護）

2. **請求先**
   - 各公費には全額請求
   - 適用範囲外の医療は下位の公費へ

3. **旭川市への請求書**
   - 12（生活保護）への請求分のみ記載
   - 21/15/16/54への請求分は記載しない
