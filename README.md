# 生活保護調剤券請求書自動作成ツール

## 概要
生活保護の調剤券に基づく請求書を自動生成するツールです。

## 機能
- 調剤券データの読み込み
- 請求書の自動生成
- 請求書のPDF出力

## 必要要件
- Python 3.8以上
- Excel 2016/2019/365（VBA版の場合）

## インストール方法
```bash
# リポジトリのクローン
git clone https://github.com/sekine53629/welfare-dispensing-invoice-generator.git
cd welfare-dispensing-invoice-generator

# 依存パッケージのインストール
pip install -r requirements.txt
```
